### Prerequisites

- Flutter & Dart SDK (https://docs.flutter.dev/get-started/install)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/station_reach.git
   cd station_reach
   ```

2. Install dependencies:
   ```bash
   flutter pub get
   ```

3. Run the app:
   ```bash
   flutter run
   ```

### Building for Production

#### iOS
```bash
flutter build ipa
```

#### Android
```bash
flutter build apk
```
